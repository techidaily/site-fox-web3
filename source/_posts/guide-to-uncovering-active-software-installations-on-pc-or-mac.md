## Detect Installed Software Dialog

 From this dialog you can add a condition which detects the software version already installed on the target machine.

![Detect Installed Software Dialog](https://cdn.advancedinstaller.com/img/dialog/detect-installed-sql.png "Detect Installed Software Dialog")  

To be more precise, a search will be performed at the beginning of the installation and the condition will be true only if one of the **bold font** items in the list are installed on the target machine. The regular font items will not be taken in consideration.

![Note](https://cdn.advancedinstaller.com/svg/common/IconMessageNote.svg)If there are several versions of the software installed on the target machine, the version selected in this dialog will be compared with the highest version from the target machine.
