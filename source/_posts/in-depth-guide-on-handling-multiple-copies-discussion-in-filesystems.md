## File Duplication Dialog

 In this dialog, you can customize a file duplication operation. Go to New File Operation and select New File Duplication ![File Duplication](https://cdn.advancedinstaller.com/img/gfx/file-duplicate.png "File Duplication"). 

![Edit File Duplication Dialog](https://cdn.advancedinstaller.com/img/dialog/edit-file-duplication.png "Edit File Duplication Dialog")  

![Caution!](https://cdn.advancedinstaller.com/svg/common/IconMessageWarning.svg)This operation can be performed only for package files.
